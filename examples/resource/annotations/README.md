# Platform Annotations Example

A self-contained example showing how to read and write Grafana platform annotations like folder
placement, manager ownership, and provisioning source off a `resource.Object` using the SDK's
exported constants and accessors instead of hard-coding annotation strings.

See [docs/annotations.md](../../../docs/annotations.md) for the full reference.

## Run

```bash
go run ./examples/resource/annotations
```
